# Glass Classification


Project based on dataset from kaggle   
https://www.kaggle.com/datasets/uciml/glass

### Techniques used in the project

+ Visualization:
    + barplot
    + correlation heatmap

+ Logaritmic Regression
+ NaiveBias
    + Gaussian NaiveBias
    + Multinomial NaiveBias
+ Decision tree
+ Grid search
+ KNNeighbours
+ Discretization
+ Scalers
    + MinMaxScaler
    + StandardScaler
    + RobustScaler
+ Metrics
    + f1_score
    + precison_score
    + recall_score
    + accuracy_score
